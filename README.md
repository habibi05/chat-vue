# vue-chat

## Project setup
### For vue
```
npm install
```
### For websocket
```
cd server
npm install
```


### Compiles and hot-reloads for development
#### For vue
```
npm run serve
```
#### For vue
```
cd server
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
